Create a Job Profile

  1. Wire up a click handler to the CREATE JOB PROFILE button such that the form can be submitted and the data is pulled from the form for processing.
  2. Every time a new job profile is submitted: the data must be displayed below the form using the example html.
    • The fields that have drop downs should have their text values displayed (ie. state should show "CA" and not "1")
    • If the user does not select anything, it should show an alert to select a state
  3. If the user does not fill a * marked field you should show an alert that tells them to fill that field
  4. After a job profile is added to the list, the form must be reset back to it's original state (i.e. all fields empty and the dropdown should says "Please Select").
  5. The LinkedIn URL should show the LinkedIn link as the example does. This is done by manipulating the href attribute of the anchor tag.
  6. Add a red delete button for each individual job profile added. When clicked this button should remove only that profile from the list.
  7. When you click anywhere on a profile in the list, you should populate the form above with the data of the selected profile. So if you were to click on the "Mark Zuckerberg" name element (anywhere in that div) the text field for "Full Name" should change to "Mark Zuckerberg".
  8. After clicking submit on the button (after you clicked on a job profile) the profile in question should be updated instead of creating new profile and adding to the list.

While we are looking for working pages, however you can make that happen, we would like to see code with:

  • JavaScript Namespacing
  • No inline functions unless absolutely required

Functional pages are the #1 priority though.
