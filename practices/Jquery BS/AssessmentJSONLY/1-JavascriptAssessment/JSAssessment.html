Create an Address

  1. Wire up a click handler to the CREATE ADDRESS button such that the form can be submitted and the data is pulled from the form for processing.
  2. Every time a new registration is submitted: the data must be displayed in the right-hand column using the example html.
    • The fields that have drop downs should have their text values displayed (ie. state should show "CA" and not "1")
    • If the user does not select a state you should show an alert that tells them to select a state
  3. After the new address is appended, the form must be reset back to it's original state (i.e. all fields empty and the dropdown should says "Please Select").
  4. The image should show the address as the example does. this is done by manipulating the src attribute of the image. So if you add an address your house, it should show your house on the map not the address that is currently shown.
  5. Add a red delete button for each individual address in the right-hand column. when clicked this button should remove only that one address from the list.
  6. When you click anywhere on an address item in the right column, you should populate the form on the left with the data in the given address. So if you were to click on the "400 Corporate Pointe" address element (anywhere in that div) the text book for "Line 1" should change to "400 Corporate Pointe".
  7. After clicking submit on the button (after you clicked on an address) the address in question should be updated instead of creating a new address in the list.

While we are looking for working pages, however you can make that happen, we would like to see code with:

  • JavaScript Namespacing
  • No inline functions unless absolutely required

Functional pages are the #1 priorty though.

The initial click handler is already provided for you in the startUp function.
