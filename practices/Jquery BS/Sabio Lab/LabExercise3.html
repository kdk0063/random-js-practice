Add Properties

Instructions

  1. Open the console in the browser and run the test sabioTester('addProperties'). You will now see the specs you need to test for. Use this function to perform tests.

  2. Create a function called sabio.page.addProperties that takes an object. Check if the parameter is not null or of type object and if so, it should add the properties name, age, gender and certificates and return the object. Certificates should be an empty array. Set the rest of the properties to null. If the parameter does not comply with the previous condition you should return "Not an object or null".

  3. Create a function called sabio.page.addId that takes an object. If the parameter is not null or of type object it should add the property id but USING bracket notation. If the parameter does not comply with the previous condition you should return "Not an object or null".
