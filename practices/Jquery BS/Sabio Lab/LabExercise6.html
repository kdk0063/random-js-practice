Conditional Statements

Instructions

  1. Open the console in the browser and run the test sabioTester('condStatement'). You will now see the specs you need to test for. Use this function to perform tests.

  2. Create a function called sabio.page.isNumber that takes a parameter and checks if the type of this parameter is an integer. If so, it should give back the bool true and if not false.(You can use the one you wrote before).

  3. Write a function called sabio.page.greaterNumber that takes two integers and returns the greater number or if both are the same return the number. Use the sabio.page.isNumber to check if the parameters entered are integers and if not you sould return the message 'Please enter integers'.

  4. Write a function called sabio.page.letterGrade that takes a number and depending on this number it converts it to the proper letter grades "A","B","C","D","F". (For example the number score of 80 would represent 80 out of a 100 and would equal a "B"). Use the sabio.page.isNumber to check if the parameter entered is an integer. If it is not an integer or if the value is not in the range of (0-100) you sould return the message 'Please enter an integer score between 0-100'.

  5. Write a function called sabio.page.pluralizeNoun that takes two parameters first should be a noun and second a number. It should make the noun lower case and return the pluralized form. Example "5 cats" or "1 dog". Make it handle the collective nouns person (should be people) and goose (should be geese).
