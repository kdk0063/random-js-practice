Find Numbers

Instructions

  1. Open the console in the browser and run the test sabioTester('loopNumbers'). You will now see the specs you need to test for. Use this function to perform tests.

  2. Create a function called sabio.page.isNumber that takes a parameter and checks if the type of this parameter is an integer. If so it should give back the bool true and if not false. (You can use the one you created in a previous exercise).

  3. Create a function called sabio.page.findNumbers that takes an array. Check if the parameter is not null or of type array and if so, it returns a new array of integers. If the parameter does not comply with the previous condition you should return 'Not an array or null'.
