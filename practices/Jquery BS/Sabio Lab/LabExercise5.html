Remove Letters

Instructions

  1. Open the console in the browser and run the test sabioTester('removeA'). You will now see the specs you need to test for. Use this function to perform tests.

  2. Create a function called sabio.page.removeA that takes a string. If the parameter is not null or of type string it should remove all the letters 'a' from the string. If the parameter does not comply with the previous condition you should return "Please enter a valid string".

  3. Create a function called sabio.page.countA that takes a string. If the parameter is not null or of type string it should look for all of the letters 'a' in the string (hint: while, make sure you check lower and upper cases). If the parameter does not comply with the previous condition you should return "Please enter a valid string".
