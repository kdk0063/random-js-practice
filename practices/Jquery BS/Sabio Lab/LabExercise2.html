Sum Function

Instructions

  1. Open the console in the browser and run the test sabioTester('sumFx'). You will now see the specs you need to test for. Use this function to perform tests.

  2. Create a function called sabio.page.isNumber that takes a parameter and checks if the type of this parameter is an integer. If so it should give back the bool true and if not false.

  3. Create a function called sabio.page.sum that takes two integers and returns the sum of these two. Use the sabio.page.isNumber function to make sure that the parameters passed to this function are integers. If the parameters are not integers the function should return the message 'Please enter an integer'.
